# Invoice Generator

A Python automation tool that generates professional PDF invoices from manual input or Excel/CSV files.

## Features
- Generate PDF invoices with automatic numbering
- Add multiple line items with quantity and price
- Import items from Excel (.xlsx) or CSV files
- Automatic total calculation
- Supports Slovak characters
- Includes due date, IBAN, and client details

## Requirements
- Python 3.9+
- reportlab
- pandas
- openpyxl

## Installation
pip install reportlab pandas openpyxl

## Usage
python3 main.py

Choose option 1 for manual input or option 2 to load from Excel/CSV.
